## 4️⃣ Step-by-Step: Add a Bed

### Step 1: Select Category
Click the dropdown → Select **"Bed Sets"**

### Step 2: Generate SKU
Click the **"Auto"** button  
→ SKU automatically fills: **BS01**

### Step 3: Enter Product Name
Type: **"Luxury King Bed with Headboard"**

### Step 4: Write Description
Type:
```
Premium quality king-size bed featuring:
- Solid wood construction
- Elegant upholstered headboard
- Available in multiple finishes
- Made to order with custom options
```

### Step 5: Upload Main Image
1. Click on the **"Click to upload main image"** box
2. Select image from your computer
3. Wait for "Uploading..." (3-10 seconds)
4. Image preview appears ✅

### Step 6: Add More Images (Optional)
1. Click on small 📷 boxes
2. Select additional angles/views
3. Click **[+]** to add more slots
4. Each image uploads individually

### Step 7: Feature Product (Optional)
Check the box **"☐ Feature on homepage"** if you want this product to appear on the homepage featured section

### Step 8: Save
Click **"Add Product"** button

✅ **Done!** Product is now live on your website!

## 5️⃣ Editing a Product

### Step 1: Find the Product
Scroll through your products in the dashboard

### Step 2: Click "Edit" Button
The same modal opens with pre-filled data

### Step 3: Make Changes
- Update name, description, category
- Replace images (click X on old image, upload new)
- Toggle featured status

### Step 4: Save Changes
Click **"Update Product"**

✅ **Changes are live immediately!**

## 6️⃣ Deleting a Product

### Step 1: Find the Product
Locate the product you want to remove

### Step 2: Click "Delete" Button
Red delete button on the product card

### Step 3: Confirm
Browser asks: "Are you sure you want to delete this product?"  
Click **"OK"** to confirm

✅ **Product removed from website!**

## 8️⃣ Image Upload Tips

### Best Practices:
✅ Use high-quality images (at least 800x800px)  
✅ Use white or neutral background  
✅ Show multiple angles  
✅ Image size: 500KB - 5MB per image  
✅ Format: JPG, PNG, WEBP  

### Main Image:
- Should be the best view of the product
- This appears on product cards in shop
- First impression for customers

### Additional Images:
- Different angles
- Close-ups of details
- Room/lifestyle shots
- Maximum 6-8 images per product

## 9️⃣ Featured Products

### What "Featured" Means:
- Product appears on **Homepage** in "Featured Products" section
- Gets more visibility
- Only feature your best/newest products

### How to Feature:
1. When adding/editing product
2. Check ☑ "Feature on homepage"
3. Save

### How Many to Feature?
- Recommended: 4-8 products
- Changes appear immediately
- Can feature/unfeature anytime

## 🔟 Common Questions

### Q: How long does image upload take?
**A:** 3-10 seconds per image (depends on file size & internet speed)

### Q: Can I change the SKU later?
**A:** Yes, but only when editing. It's better to keep original SKU for consistency.

### Q: What happens when I delete a product?
**A:** It's removed from:
- Shop page
- Search results
- Homepage (if featured)
- Customer carts/wishlists

### Q: Can I undo a delete?
**A:** No, deletion is permanent. You'll need to add the product again.

### Q: How do I logout?
**A:** Click "Logout" button in top-right corner

### Q: What if I close the browser?
**A:** Your session is saved. Next time you open admin URL, you'll still be logged in (for 24 hours).

## 1️⃣4️⃣ Troubleshooting

### Problem: "Failed to connect to server"
**Solution:**
- Backend server must be running
- Check: http://localhost:3000/api/health
- Restart backend: `npm start` in backend folder

### Problem: "Image upload failed"
**Solution:**
- Check internet connection
- Try smaller image (<5MB)
- Check image format (JPG/PNG/WEBP)
- Refresh page and try again

### Problem: "Invalid credentials"
**Solution:**
- Email: `tfgallery0@gmail.com` (exact)
- Password: `TFG@2026Secure` (case-sensitive)
- No extra spaces

### Problem: Product not showing on website
**Solution:**
- Refresh website (Ctrl+F5)
- Check if product was actually saved
- Check browser console for errors
