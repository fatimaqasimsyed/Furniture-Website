## 🚀 Get Started in 3 Steps (5 Minutes)

### 1. Install Backend
```bash
cd backend
npm install
```

### 2. Start Backend
```bash
npm start
```

### 3. Start Frontend (New Terminal)
```bash
npm run dev
```
